# Ubuntu Server + CasaOS Setup

## Overview

This project demonstrates the setup of a **home server** using **Ubuntu Server**, **CasaOS** and  to host services and manage applications through a web interface.

## Features

* Ubuntu Server installation and configuration
* CasaOS for web-based server management
* Firewall and port forwarding configuration
